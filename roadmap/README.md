# 学习路线

> 如果你不想随机阅读，而是希望把知识库组织成一条可以执行的路线，就从这里开始。

## 三种使用方式

| 你现在的状态 | 建议入口 | 说明 |
|--------------|----------|------|
| 先看全局地图 | [00-overview.md](00-overview.md) | 先建立知识图谱，再决定先学哪里 |
| 想做完整冲刺 | [01-12week-plan.md](01-12week-plan.md) | 适合系统化复习或转岗准备 |
| 想立刻执行 | [02-45day-daily-plan.md](02-45day-daily-plan.md) | 给出更细颗粒度的日程拆分 |

## 推荐搭配

- 每周主线内容从 [../notes/README.md](../notes/README.md) 进入。
- 公式速查用 [../math_dictionary/README.md](../math_dictionary/README.md)。
- 对应代码与实验用 [../src/README.md](../src/README.md) 和 [../notebooks/README.md](../notebooks/README.md)。
- 面试表达与项目复盘用 [../mock_interview/README.md](../mock_interview/README.md)。
