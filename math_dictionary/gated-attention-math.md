# 门控注意力（Gated Attention）速查

## 1) 核心思想
- 在注意力输出或通道上加 gate，让模型动态控制信息流。
- 常见形式：
  `Y = g(X) ⊙ Attn(X) + (1-g(X)) ⊙ R(X)`
  其中 `g(X)=sigmoid(W_g X)`，`R(X)` 可是残差或线性分支。

## 2) 作用
- 抑制噪声 token/通道，增强重要信息路径。
- 在长上下文或检索增强场景常用于稳定信息融合。

## 3) 代价
- 额外参数和算子开销（gate 分支）。
- 若 gate 饱和（接近0或1），可能造成梯度与表达受限。

## 4) 与 KV 系统问题关联
- 可把 gate 得分作为“块价值”先验，辅助压缩/驱逐优先级。
- 线上要验证 gate 与延迟开销是否值得。

## 5) 面试一句话
- “门控注意力是可学习的信息阀门，本质是‘让模型决定看多少、走哪条路’。”
