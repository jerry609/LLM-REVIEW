# 模拟面试题库

> 按主题、公司风格和行为问题组织的面试训练材料。为了更适合 GitBook 阅读，这里改成了“入口清晰、使用场景明确”的结构。

## 三个入口

| 场景 | 建议入口 | 适合什么时候用 |
|------|----------|----------------|
| 系统刷题 | [by-topic/README.md](by-topic/README.md) | 面试前集中按模块查缺补漏 |
| 模拟真实面试官风格 | [by-company/README.md](by-company/README.md) | 针对具体公司做冲刺 |
| 整理项目故事与行为题 | [behavior/README.md](behavior/README.md) | 准备 STAR、项目复盘、冲突处理 |

## 推荐配合方式

- 先用 [../math_dictionary/README.md](../math_dictionary/README.md) 把公式与量级过一遍。
- 再去 [by-topic/README.md](by-topic/README.md) 按主题刷题。
- 最后用 [behavior/project-one-pager.md](behavior/project-one-pager.md) 把项目经历整理成可讲述版本。

## 常用补充入口

- [behavior/project-one-pager.md](behavior/project-one-pager.md)：把复杂项目压成一页纸。
- [../roadmap/README.md](../roadmap/README.md)：如果你需要按周推进刷题计划。
- [../benchmarks/README.md](../benchmarks/README.md)：记录热点和阶段性复盘。
